List of programmers

From Wikipedia, the free encyclopedia

This is a list of programmers notable for their contribution to software, either as as original author or architect, or for later additions. All entries must already have associated articles. Some persons notable as computer scientist are included here because they work in program as well as research.

NOTE: This is a dynamic list and may never be able to satisfy particular standards for completeness. You can help by editing this page to add missing items, with references to reliable sources.
