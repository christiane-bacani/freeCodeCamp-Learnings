Browser Extension Installation Guide for daily.dev

The daily.dev browser extension is a powerful tool to enhance your daily.dev experience. It allows daily.dev to show up in a new tab, helping you develop a reading habit, You can pause the new tab behavior on daily.dev at any time. Additionally, the extension provides access to the companion widget for daily.dev for added convenience.

Supported Browsers for daily.dev

You can install the daily.dev extension on the following browsers:
  • Google Chrome Extension
  • Microsoft Edge Extension
